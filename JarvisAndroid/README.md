# J.A.R.V.I.S. Android (Premium Flutter Edition) 🚀🤖

Это полнофункциональная архитектура для Android-версии J.A.R.V.I.S., построенная на Flutter. Проект переработан для обеспечения максимальной производительности и визуального совершенства.

## Что нового в этом обновлении:

1.  **Архитектура сервисов**:
    - `GeminiService`: Реализует последовательную потоковую передачу данных (Sequential Streaming) через WebSocket.
    - `AudioService`: Настроен на захват чистого PCM 16kHz звука.
    - `TTSService`: Реализует Jitter Buffer с мгновенным стартом (отклик < 1 сек).
2.  **Премиальный UI**:
    - **Glassmorphism**: Использование блюр-эффектов и полупрозрачных панелей.
    - **Анимации**: Пульсирующая кнопка записи и плавные переходы.
    - **Цвета**: Полное соответствие iOS версии (Emerald / Dark Navy).
3.  **Управление состоянием**: Использование `Provider` для стабильной работы интерфейса.

## Как запустить:

1.  **Настройка SDK**: 
    - Убедитесь, что у вас установлен Flutter SDK.
    - В корне папки `JarvisAndroid/android/` создайте файл `local.properties`.
    - Добавьте: `flutter.sdk=/путь/к/вашему/flutter`
2.  **Зависимости**:
    - В терминале внутри `JarvisAndroid` выполните: `flutter pub get`.
3.  **Android Studio**:
    - Откройте папку `JarvisAndroid` (значок Flutter).
    - Выберите устройство и нажмите "Run".

## Структура проекта:
- `lib/services/`: Логика работы с ИИ и звуком.
- `lib/ui/`: Экраны и компоненты интерфейса.
- `lib/state/`: Глобальное состояние приложения.

**Важно**: Ваши ручные правки в файлах `build.gradle` и `AndroidManifest.xml` сохранены и учитываются при сборке.
